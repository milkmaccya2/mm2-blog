## 今週のハイライト

今週はブログの大幅改修フェーズと言える一週間だった。Astroベースの自作ブログに、PageFindによる検索機能やTransition、ダークモード対応などを実装。あわせてAI（Gemini/Claude）を駆使した開発フローも試せて、技術的な手応えが感じられた。家族イベントや育休復帰を見据えた生活環境の整備も進み、公私ともに密度の高い時間を過ごせた。

## 家族・生活

### サンリオピューロランドと「鍵」のハプニング

家族でサンリオピューロランドへ。友人と行く予定だったが、あいにく体調不良とのことで、急遽ばあばに双子の子守りをお願いしての出陣となった。お目当てのポムポムプリンには会えなかったが、キティちゃんをはじめとするキャラクターたちに囲まれ、長女（2歳）も満足げだったのが救いだ。

ただ、出発時に焦って鍵を閉めてしまい、隣人に迷惑をかけてしまうという手痛いミスも。翌日、幼稚園の書類提出のついでに、サウナ帰りに家へのお土産として池袋で購入したニューヨークチーズケーキとピューロランドで買ったお菓子を持って挨拶に伺った。こうしたトラブルへのリカバリーも、育休中の身としては大事な経験値（？）だ。

### 長女の発熱と寝室運用の変更

週の後半、長女が保育園で38℃近い熱を出した。たまたま出先で迎えに行きやすかった妻に迎えをお願いし、現在は徐々に熱も下がって落ち着いている。この発熱を受け、予定していた「家族全員で同じ部屋で寝る」という寝室運用の変更は再来週まで保留することにした。長女にとっては初めてのベッド生活になるため、少し心配ではあるが、まずは体調第一。育休復帰に向けて、焦らずリズムを整えていきたい。

### 筋トレとサウナでのリフレッシュ

パーソナルトレーニングにて、ベンチプレス55kgを10回成功。着実にパワーが戻ってきている。トレーニング後は丸亀製麺で炭水化物を補給し、タイムズスパレスタでサウナと仮眠を堪能。この「追い込みと癒やし」のセットが、今の自分の貴重なメンタルメンテナンスになっている。

### Chrome DevTools MCP & Playwright MCP

[Chrome DevTools MCPを触ってみる](https://github.com/google/chrome-devtools-mcp)
Node.jsやChromeの最新版が必要だが、E2Eテストに活用できるポテンシャルを感じる。Playwright MCPとの使い分けについては、両方触りながら検証していく。

### iPadからの開発環境構築（Termius + Tailscale）

[iPadからmacに接続してgemini cliを試行](https://tailscale.com/kb/1017/install-ipad)
TailscaleとTermiusを組み合わせることで、モバイル環境からのコーディング体制が整いつつある。場所を選ばず「スマホバイブコーディング」ができるのは熱い。

### Docusaurusによるドキュメント整備

[hostswitchのドキュメント作成](https://docusaurus.io/)
簡単に美しいドキュメントが生成できる。ブログ側にもこうしたドキュメント管理の仕組みを統合していきたい。

### macOSの操作環境改善

[Rectangle](https://rectangleapp.com/) / [AltTab](https://alt-tab-macos.io/)
Windowsのようなウィンドウ操作感をmacで実現。スイッチしたばかりの身としては、この設定だけで作業効率が爆上がりした。

## 振り返り

今週はとにかくブログのパフォーマンス、デザイン、機能面を徹底的に磨き上げた。特に、Gemini Code AssistとAntigravityのSKILLSを組み合わせたPRレビューから修正への流れは非常にスムーズで、開発体験が格段に向上した。Antigravityはモデルを **Claude 4.5 Sonnet** に切り替えたことで、コーディングの推論能力がさらに一段上がった実感が強い。

また、Notionで書いたメモをGeminiでMarkdown化し、Astroに反映してGitHubにPushするというパイプラインが完成したのも大きい。さらに **nano banana pro** を使ったヒーロー画像の生成など、AIツールをフル活用してブログの完成度を上げられたのは、大きな収穫だった。

来週は長女の体調を見守りつつ、育休復帰に向けた生活リズムの微調整を継続する。投資信託のNISA成長投資枠への移行など、資産運用のメンテナンスも進められたので、足元を固めつつ、攻めの学習も継続していきたい。
