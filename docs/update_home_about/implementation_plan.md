# ホームとアバウトページの更新計画

## 目標
Astroのデフォルトテンプレートのままになっているホーム (`index.astro`) とアバウト (`about.astro`) ページを、ユーザー自身の情報と職歴書（CV）スタイルの内容に更新する。

## ユーザーレビューが必要な事項
- `about.astro` の具体的な職歴データはプレースホルダーで作成するため、後でユーザー自身に入力してもらう必要がある。
- `index.astro` の自己紹介文も同様。

## 提案される変更

### ページ構成の変更

#### [MODIFY] [`src/pages/index.astro`](file:///Users/yokoyama/git/mm2-blog/src/pages/index.astro)
- デフォルトのAstro紹介テキストを削除。
- 以下のセクションを追加:
    - **ヒーローセクション**: 簡潔な挨拶と自己紹介。
    - **最新のブログ記事**: `getCollection('blog')` を使用して、最新の記事（例: 3〜5件）をリスト表示する。

#### [MODIFY] [`src/pages/about.astro`](file:///Users/yokoyama/git/mm2-blog/src/pages/about.astro)
- デフォルトのLorem Ipsumテキストを削除。
- 職歴書（CV）スタイルのレイアウトに変更:
    - **プロフィールヘッダー**: 名前、肩書き、SNSリンク。
    - **職歴 (Experience)**: 時系列での職歴リスト。
    - **スキル (Skills)**: 技術スタックのリスト。

## 検証計画

### 自動テスト
- 現状テストコードはないため、ビルドが通ることを確認する。
- `npm run dev` でエラーが出ないことを確認する。

### 手動検証
- ブラウザプレビューで以下の点を確認する:
    - ホームページに最新の記事が表示されているか。
    - アバウトページがCVスタイルで正しく表示されているか。
    - レスポンシブデザイン（モバイル表示）が崩れていないか。
