# ブログ一覧ページネーション 実装結果

## 概要
ブログ一覧ページのページネーション機能を実装しました。
また、PR レビュー指摘事項に基づき、コードのリファクタリング（コンポーネント化）とアクセシビリティ改善（画像代替テキストの強化）を実施しました。

## 変更内容詳細
### 1. 定数の追加
- **ファイル**: `src/consts.ts`
- **内容**: ページネーションサイズを定義する `BLOG_PAGE_SIZE`（値: 10）を追加。

### 2. 記事リストコンポーネントの作成
- **ファイル**: `src/components/BlogPostList.astro` (新規作成)
- **目的**: 記事リスト表示ロジックの共通化と再利用。
- **機能**:
    - `posts` Props (CollectionEntry<'blog'>[]) を受け取り、記事一覧をレンダリング。
    - 各記事にリンク、ヒーロー画像、タイトル、日付を表示。
    - **改善**: ヒーロー画像の `alt` 属性に `${post.data.title}のヒーロー画像` を採用し、アクセシビリティを向上。

### 3. 動的ページネーションページの実装
- **ファイル**: `src/pages/blog/[page].astro` (新規作成)
- **機能**:
    - `getStaticPaths` で全記事を取得し、`paginate` 関数を使用してページごとのデータを生成。
    - `BlogPostList` コンポーネントを使用し、各ページの記事を表示。
    - 「前へ」「次へ」およびページ番号（例: 2 / 5）を表示するナビゲーションを追加。

### 4. ブログトップページの修正
- **ファイル**: `src/pages/blog/index.astro`
- **変更点**:
    - 全記事を取得後、最初の `BLOG_PAGE_SIZE` 件のみを表示するように変更。
    - `BlogPostList` コンポーネントを使用。
    - 記事数がページサイズを超える場合、「次のページ →」リンク（`/blog/2/`）を表示。
