GSAPを用いたホームページの改修

## 概要
GSAP (GreenSock Animation Platform) を導入し、トップページ（`src/pages/index.astro`）のデザインとアニメーションを大幅に強化しました。
参考サイトの雰囲気に近づけるため、ブロックリビールエフェクト（塗りつぶしてから文字が出る演出）やスクロール連動のアニメーションを実装しました。

## 実施した作業
1.  **依存関係の追加**
    *   `gsap` パッケージをインストール。
2.  **HTML/CSS 構造の変更**
    *   タイトルやサブタイトルを `reveal-wrap` で囲み、アニメーション用の `reveal-block` 要素を追加。
    *   背景に動的なグラデーションブラーを追加（`animate-blob`）。
    *   記事リストのレイアウト調整と、モダンなカードデザインへの変更。
3.  **アニメーション実装 (GSAP & ScrollTrigger)**
    *   `gsap.timeline` を使用し、ブロックが拡大→文字表示→ブロック縮小という一連の流れを作成。
    *   `ScrollTrigger` を設定し、要素が画面内に入ったタイミング（`start: "top 90%"`）でアニメーションを発火。
    *   スクロールバック時の動作制御（`toggleActions`）。
    *   ホバー時の要素インタラクション（拡大、影の強調）。

## ファイルの変更点
- `package.json`: `gsap` の追加。
- `src/pages/index.astro`: レイアウトの刷新、GSAPスクリプトの追加、スタイル定義の更新。

## 確認事項
- `npm run dev` で起動し、トップページのアニメーションがスムーズに動作すること。
- スクロールした際に「Recent Posts」などがふわっと表示されること。
- ダークモードでも視認性に問題がないこと。
