# 変更点の確認: ブログ記事のナビゲーション追加

以下の変更を行い、ブログ記事ページにナビゲーションリンクを追加しました。

## 変更ファイル

### 1. `src/pages/blog/[...slug].astro`
- **ロジックの変更**:
  - `getCollection('blog')` で取得した記事を `pubDate` の降順でソートするように変更しました。
  - 各記事のパスを生成する際に、`prevPost`（ひとつ古い記事）と `nextPost`（ひとつ新しい記事）を計算し、`props` に含めるようにしました。
  - コンポーネントのプロパティ受け渡し部分を更新し、`BlogPost` レイアウトへ `prevPost` と `nextPost` を渡すようにしました。
- **クリーンアップ**:
  - 未使用の `CollectionEntry` インポートを削除しました。

### 2. `src/components/BlogNavigation.astro` (新規作成)
- **共通コンポーネント化**:
  - ナビゲーションリンクのマークアップを独立したコンポーネントに切り出しました。
- **レスポンシブ対応 (モバイル横並び)**:
  - 元のグリッドレイアウトから `flex` レイアウトに変更し、モバイル画面でも常に横並びになるようにしました。
  - 画面幅が狭い場合でもレイアウトが崩れないよう、以下の工夫を行いました：
    - テキストの短縮: モバイルでは「前の記事」を「前へ」のように短く表示。
    - 省略表示の強化: `min-w-0` と `truncate` を組み合わせて、長いタイトルが適切に省略されるように設定。

### 3. `src/layouts/BlogPost.astro`
- **コンポーネントの導入**:
  - 新規作成した `BlogNavigation` をインポートして使用するように変更しました。
- **配置の追加**:
  - ページ下部だけでなく、**ページ上部（タイトルの下、目次の上）**にもナビゲーションを追加しました。
- **コード削減**:
  - 以前の直書きHTMLを削除し、コンポーネント呼び出しに置き換えました。


## 確認方法
- 任意のブログ記事ページにアクセスし、ページ下部にナビゲーションが表示されていることを確認してください。
- 最新の記事および最古の記事では、それぞれ「次の記事」や「前の記事」が表示されないことを確認してください。
- 長いタイトルの記事へのリンクが省略表示されることを確認してください。
