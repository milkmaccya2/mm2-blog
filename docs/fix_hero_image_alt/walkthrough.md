# 変更内容の確認: Hero画像のalt属性設定

以下のファイルに対して変更を行い、Hero画像の `alt` 属性に記事タイトルを設定しました。

## 変更ファイル

### `src/layouts/BlogPost.astro`
- Hero画像の `alt` 属性を `title` から `` `${title}のアイキャッチ画像` `` に変更しました。これは `h1` タグとの読み上げ重複を避けるためです。

### `src/pages/blog/index.astro`
- 記事一覧のサムネイル画像の `alt` 属性を `alt=""` に戻しました。リンク全体としてテキスト（記事タイトル）が含まれており、スクリーンリーダーでの冗長な読み上げを防ぐためです。

## 意図
- アクセシビリティの向上とSEO対策のため、意味のある `alt` テキストを設定しました。
