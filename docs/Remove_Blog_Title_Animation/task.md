# タスク: ブログ記事タイトルのスクランブルアニメーション削除

## 目的
ブログ記事ページにおいて、タイトルが読み込まれるときのアニメーション（Scramble Text）を削除する。

## 背景
- タイトルが長い場合、アニメーション中にレイアウトシフト（CLS）が発生してしまう。
- ユーザーにとって読みづらく、UXを損なう可能性がある。

## 要件
1. ブログ記事ページ（`src/layouts/BlogPost.astro`）のタイトル部分（`h1`）から、`scramble-text` クラスを削除する。
2. アニメーションに関連する `data-original-text` 属性も削除する。
3. 静的なテキストとしてタイトルを表示する。
