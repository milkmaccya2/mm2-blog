# タスク: ホームページのProjectsとRecent PostsにScrambleText効果を追加する

## 目的
ユーザーのリクエストに基づき、ホームページの「Projects」および「Recent Posts」のテキストに、ランダムな文字が入れ替わりながら最終的なテキストが表示される「ScrambleText」アニメーションを実装する。

## 要件
1.  **対象要素**:
    - Projectセクションのタイトル
    - Recent Postsセクションのタイトル
2.  **アニメーション効果**:
    - スクロールに合わせて発火 (ScrollTrigger)。
    - ランダムな文字列から徐々に元のテキストに変化する。
    - `prefers-reduced-motion` が有効な場合はアニメーションをスキップする。
3.  **技術スタック**:
    - GSAP (既存のものを使用)
    - カスタムScrambleText実装 (プラグインなし)

## 完了条件
- ホームページをスクロールすると、対象のタイトルがスクランブル効果で表示されること。
- View Transitions後も動作すること。
- エラーが発生しないこと。
